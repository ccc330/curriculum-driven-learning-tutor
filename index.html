📌 部署说明

  1. 确保已安装Python 3.7+环境
  2. 安装项目依赖：pip install -r requirements.txt
  3. 配置环境变量：复制 .env.example 为 .env 并填入您的API密钥
  4. 启动后端服务：python backend_api.py
  5. 在浏览器中打开 learning_tutor_app.html 或点击上方按钮
